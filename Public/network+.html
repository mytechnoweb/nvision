Key Topics & Definitions

What is Networking?

Networking is the practice of connecting computers and other devices to share resources and information.

OSI Model

The OSI (Open Systems Interconnection) model is a conceptual framework used to understand network interactions in seven layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.

IP Address

An IP address is a unique identifier assigned to each device on a network. It can be IPv4 (e.g., 192.168.1.1) or IPv6 (e.g., 2001:0db8::1).

Subnetting

Subnetting divides a network into smaller, manageable segments. It improves performance and security by isolating traffic.

TCP vs UDP

TCP (Transmission Control Protocol) is connection-oriented and reliable. UDP (User Datagram Protocol) is connectionless and faster but less reliable.

Firewall

A firewall is a security system that monitors and controls incoming and outgoing network traffic based on predetermined rules.

VPN (Virtual Private Network)

A VPN creates a secure, encrypted connection over a less secure network, such as the internet, allowing private data to be transmitted safely.

MAC Address

A MAC (Media Access Control) address is a hardware identifier for network interfaces, used in the Data Link layer of the OSI model.
